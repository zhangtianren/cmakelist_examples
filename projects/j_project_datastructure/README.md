# 数据结构

## 1、线性存储结构
##### 1.1 单向链表，双向链表
##### 1.2 顺序栈，链式栈
##### 1.3 队列，双端队列

## 2、树形存储结构
##### 1.1 普通树
##### 1.2 二叉树，森林
##### 1.3 平衡二叉树
##### 1.4 应用: 先序、中序、后序、层序遍历树；有序树的查找；avl的插入删除查找；hufman树的构建与hufman编码

## 3、图的存储
##### 1.1 邻接矩阵图，邻接表图
##### 1.2 遍历：深度优先，广度优先
##### 1.3 应用：最小生成树，最短路径，拓扑排序，关键路径
